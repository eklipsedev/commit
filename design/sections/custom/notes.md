Custom sections are composed in Sanity as a single `customSection` block with nested modules (tagline, headline, split, text grid, lists, attributes, steps, button).

**Split row:** pick 1 or 2 columns, then compose each column from the same modules as the section (except nested split rows). Example: string list on the left, headline on the right.

**Templates:** Config → Flexible section templates. When inserting a Flexible section, choose **Start blank** or **From template…**. Templates are copied onto the page (not linked), so each page can diverge.

Reuse:
- `textGrid` — also used in custom section modules and split columns
- `detailAttributes` — also used on offering overlays
