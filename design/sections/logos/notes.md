There are 2 main variants here:
- full width
- limited

Full width goes end to end of the screen and scrolls. If there are not enough logos, it will duplicate the logos that do exist so there are enough to scroll

The scroll effect should use CSS, not JavaScript

If the limited option is set, it will accept up to 6 logos across before it begins to wrap. Logos should remain consistently spaced and sized appropriately visually

Needs to be mobile responsive or potentially use container queries to make it more flexible
